# Category photos

Real photos for the Discover-section event cards (the header image on each
event card, picked by the event's category). Each file below replaces one
existing illustration — if a file is missing, that card automatically falls
back to its current placeholder illustration, so you can add these one at a
time.

| File | Used for category | Currently shows |
|---|---|---|
| `conferences.jpg` | Conferences | `main-stage.svg` illustration |
| `concerts.jpg` | Concerts | `concert-crowd.svg` illustration |
| `workshops.jpg` | Workshops | `workshop-room.svg` illustration |
| `weddings.jpg` | Weddings | `marquee-gardens.svg` illustration |

Recommended spec: landscape crop (roughly 16:9 or wider), at least 800×450, JPG, under ~400KB each.
