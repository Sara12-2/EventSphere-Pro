# Role card photos

Small circular photos shown on the three role cards in the "Three roles, one
platform" section. If a file is missing, that card just shows its icon badge
instead — nothing breaks.

| File | Card |
|---|---|
| `role-admin.jpg` | Admin |
| `role-organizer.jpg` | Organizer |
| `role-attendee.jpg` | Attendee |

Recommended spec: square crop (1:1), at least 200×200, JPG, under ~150KB each.
